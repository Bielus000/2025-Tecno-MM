Projecte `working-js-00`

Contingut: una estructura mínima per començar amb HTML/CSS/JS.

Fitxers i carpetes:
- `index.html` - pàgina principal
- `css/main.css` - estils
- `js/app.js` - script bàsic
- `img/` - carpeta per imatges (conté `.gitkeep`)

Com provar-ho:
1. Obre `index.html` al navegador (doble clic o arrossega al navegador).
2. Pots servir-ho amb un servidor estàtic si cal (per ex. `npx http-server` o `python -m http.server`).

Notes:
- Pots canviar els noms si prefereixes `styles.css` o altra estructura.
